# Echo Bot 功能


#### 訂閱午餐外送提醒

```javascript
開啟訂餐通知    // 訂閱此功能
關閉訂餐通知    // 取消訂閱此功能
```



#### 點餐收集功能

```javascript
開始點餐    // 啟動功能
結束點餐    // 結束功能

// 在功能啟動時才有效的指令
清單       // 列出目前已收集的訂單
/餐點名稱   // 收集餐點
```



#### 每月IMS回報工時提醒功能

```javascript
啟動IMS回報    // 訂閱此功能
關閉IMS回報    // 取消訂閱此功能
```
